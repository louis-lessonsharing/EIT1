Ecole-it premiéres années

Description de la formation

Cette formations est pour des élèves apprennant les bases de l'informatique. Les élèves en première année se preparent pour passer un BTS SIO - Service informatique aux Organisations, se former en deux ans aux métiers d’administrateur réseau ou de développeur pour intégrer directement le marché du travail avec le diplome valent un bac +2, ou les élèves peuvent continuer des études dans le domaine de l’informatique. Ce qui veulent continuer dans le domaine feront un total de trois années pour avoir un bachelier, ou de cinq années pour avoir un master.

Competences

  • commandes de bases linux
  • composant ordinateur
  • development de la presence en ligne de l'organisation
  • estimations des resources pour une tache
  • instalation d'un serveur web sur linux
